## Run Locally

Clone the project

```bash
  git clone https://github.com/MarceeloDominguez/Shopping-cart-react-native.git
```

Go to the project directory

```bash
  cd Shopping-cart-react-native
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
